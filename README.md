# Listit

Empty shell for a fresh stack. From the repo root:

1. **TanStack Start** — follow [TanStack Start — Getting Started](https://tanstack.com/start/latest/docs/framework/react/getting-started) (for example `npm create @tanstack/start@latest` in this directory, or create a subfolder and move files as the wizard suggests).

2. **Convex** — after the app exists, run `npx convex dev` from the project root (or the package root Convex should use) and complete [Convex quickstart](https://docs.convex.dev/quickstart) for your framework.

Keep `.env.example` aligned with whatever auth and API keys you add.
